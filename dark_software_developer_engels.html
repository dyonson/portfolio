Software Developer

As a software developer, you design, develop and maintain
software programs, such as apps, websites and systems. You
write code in various programming languages, test and
debug the software to fix errors. In addition, you collaborate
with other team members, such as designers and testers,
and document the code for later use. You also ensure that
existing software is maintained and improved. In short, you
are responsible for creating and optimizing software that
meets the needs of users.
